EBM Installation Guide

Client: Shakespeare & Co.

Project: Engineer Installation Guide

Date: 2019

The Espresso Book Machine (EBM) is a book printer which allows bookstore customers to have their own PDFs, or those from a catalog, printed on-demand into bound hard copies. I was hired on to update legacy code to run on modern MacOS, as well as to document the existing codebase. This guide is meant for future software engineers who are comfortable with command-line but may not be used to the EBM, MacOS, or the languages and plugins used by the system.

This guide was originally written in Markdown.
